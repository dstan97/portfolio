In the section above I have created two data bindings. The first 'Linkedin' link is using a data attribute called 'website' that is stored in the JavaScript code for this page. The 'website' data attribute is a variable with its data being the 'Linkedin' URL.

The second 'Linkedin' link is similar to the first, except that the 'a href' tag is also stored in the data attribute.

The name & job attributes are just simple variables containing text strings.

Events:

My age is {{age}}

{{x}}, {{y}}

The age attribute is a simple variable with a value of 22. The add/subtract buttons use an increment function and a decrement function that are both bound to click events. You can use these buttons to add or subtract a year from the age.

The add 10 & subtract 10 buttons work in the same way except they increment and decrement by 10 and are both bound to double click events.

The canvas box on the right tracks the users mouse position when they hover over it, by showing the co-ordinates. This is done by using X & Y variables that are then offset when the user hovers the mouse over the canvas.

The link in the modifier section above has a 'prevent' function attached to its click event which stops the left click from sending the user to the URL. Instead the user is given an alert suggesting that they open the link in another tab, as the 'prevent' function is only applied the left click.

The lists above contain information about characters from the game series 'Tekken'.

The data is displayed through a for loop adding each list item to a unordered list in the first section and a template in the second section.

The data is displayed in a 'key' - 'value' format in the 3rd section.

Destroy The Car!

The final section includes a simple game where you click the 'punch' button to lower the car's health until it is destroyed.

The user can also restart the game which will reset the car's health bar and reset the image of the car.

The 'punch button works by removing 10 from the car's health each time the button is clicked until the healthbar reaches 0. The 'punch button then disappears and teh car image is replaced with a destroyed car.
